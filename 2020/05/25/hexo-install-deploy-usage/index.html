3. 使用

  1. 添加文章的2种方法

    • hexo命令hexo new "文章标题"
    • 手动编辑md文件，放入source/_post 文件夹

  2. 修改(或删除)文章

    • 进入source/_post文件夹，删除(或删除)post_title.md文件

  3. 添加、修改、删除后

    • 执行hexo g、hexo d即可
